# 📚 Book Management System (Java)

## 📌 Overview

A **console-based Java application** to manage books with features like adding, viewing, searching, updating, deleting, and sorting.

This project is built in **multiple versions (v1 → v4)** to demonstrate incremental development and strong understanding of core Java concepts.

---

## 🚀 Features (Overall)

* Add books
* View all books
* Search books (ID, Title, Author)
* Compare book prices
* Update book details
* Delete books
* Sort books (Title → Price)
